### Usage:\
* `MVC`\
* `Layout Constraints`\
* `Firebase Database`\
* `Firebase Email Auth`\

* support all iPhone models\
* Add, create user in Firebase DB\
* Check and inform in case of bad email input\
* Keyboard appearence will not overlap the login screen\
* ToDo list has completed/incompleted options\
